# Business CRM & Operations Management System

A multi-tenant CRM built with Next.js, Supabase, and Tailwind CSS.

## Overview
This project is a multi-tenant CRM designed for small businesses to manage customers, deals, and tasks.

## Tech Stack
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (Auth, Postgres, RLS)
- **Deployment**: Vercel
